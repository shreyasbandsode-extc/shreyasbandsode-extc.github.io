### 🌱 Soil Moisture Monitoring System (Solar Powered)
> *Sustainable IoT for smart agriculture*

An autonomous soil moisture monitoring system powered by solar energy — monitors moisture levels and triggers irrigation when needed, running completely off-grid.

- **Sensor:** FC-28 soil moisture sensor
- **Power:** Solar panel + battery backup
- **Controller:** Arduino
- **Key concept:** Renewable energy integration, analog sensor reading
- 📁 [View Project →](https://github.com/shreyasbandsode-extc/soil-moisture-monitoring-system)
